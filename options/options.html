Log Folder

Select the folder where PlexComponentHost writes its log files. The extension reads files directly from disk — no server or Python required.

Enter the label dimensions in inches (must match your physical label stock).

Download Folder

Subfolder within your browser's default Downloads directory where rendered labels will be saved. Leave blank to save directly to Downloads.

Example: ZPL Labels → saves to ~/Downloads/ZPL Labels/label-xxx.png

Polling Interval

How often to check the log file for new print jobs. Note: Chrome enforces a minimum of 30 seconds for background polling. The popup checks immediately when opened.

Options

Preview Pop-up

Automatically show a floating preview window when a new print job is detected. The window closes itself after the configured time.
